# NumpadEmu
Little numpad emulator originally made for myself to use numpad buttons on laptops and keyboards without numpad. Made for win11 x64

How to use:
"Pause Break" — turn Numpad on/off
While Numpad on:
"=" — NumLock on/off
While Numlock off — 1-0 for arrows, page up, page down, home, insert...
While Numlock on — left alt + numbers allow you to use alt-codes. Sadly, numbers don't work in this mode :(

Program works in background and can be closed via task manager

That's all for now :)
